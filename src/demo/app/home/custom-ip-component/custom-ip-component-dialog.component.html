The NgxIp component is using a theme to apply input layout material style input design.

If your app is already using the @angular/material and @angular/flex-layout libraries we can create a custom control that uses the native MatInput component from material and fxFlex directives for layout.

By extending the base component class (NgxIpBase) custom implementation can be set.

In the following example, a custom component is used to implement angular/material Input component with angular/flex-layout components. This implementation has no CSS and is using the full power of the flex-layout and material libraries.

This is not restricted to material / flex-layout, you can customize with any libarary or custom element you want.
